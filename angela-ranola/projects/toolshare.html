ToolShare

ToolShare is an Android application where users can rent out tools, as well as rent others' tools. It was developed using Android Studio and has a SQLite database.

With the app, users are able to:

  • Add tools to the application
  • Post ads for their tools
  • Request to rent tools
  • Respond to rent requests (accept, reject)

Developed with two other developers, my work for the application included user interface design, a large portion of the backend logic, as well as all database work.
